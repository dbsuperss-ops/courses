# 실전 프롬프팅 (Real World Prompting)

Anthropic의 종합 실전 프롬프팅 튜토리얼에 오신 것을 환영합니다. 이 과정은 이미 프롬프트 엔지니어링의 세계에 발을 들였고, 특히 우리의 종합 **[프롬프트 엔지니어링 대화형 튜토리얼](../prompt_engineering_interactive_tutorial/README.md)**을 수료한 숙련된 개발자들을 위해 설계되었습니다. 아직 해당 튜토리얼을 거치지 않으셨다면, 다양한 프롬프팅 기술을 실습과 함께 심층적으로 다루고 있으므로 계속 진행하기 전에 완료하시는 것을 강력히 권장합니다.

5개의 레슨을 통해 주요 프롬프팅 기술을 복잡하고 실제적인 프롬프트에 통합하는 방법을 배우게 됩니다. 각 레슨은 이전 레슨에서 가르치는 핵심 개념을 바탕으로 하므로, [프롬프팅 요약 (Prompting recap)](./01_prompting_recap.ipynb) 레슨부터 시작하는 것을 권장합니다.
