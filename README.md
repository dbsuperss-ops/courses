# Anthropic 교육 과정 (Anthropic Courses)

Anthropic의 교육 과정에 오신 것을 환영합니다. 이 저장소에는 현재 다섯 개의 과정이 포함되어 있습니다. 과정을 다음 순서대로 완료하는 것을 권장합니다:

1. [Anthropic API 기초 (Anthropic API fundamentals)](./anthropic_api_fundamentals/README.md) - Claude SDK 작업의 필수 사항인 API 키 발급, 모델 파라미터 활용, 멀티모달 프롬프트 작성, 스트리밍 응답 등을 다룹니다.
2. [프롬프트 엔지니어링 대화형 튜토리얼 (Prompt engineering interactive tutorial)](./prompt_engineering_interactive_tutorial/README.md) - 주요 프롬프팅 기술에 대한 종합적인 단계별 가이드입니다. [[AWS 워크숍 버전](https://catalog.us-east-1.prod.workshops.aws/workshops/0644c9e9-5b82-45f2-8835-3b5aa30b1848/en-US)]
3. [실전 프롬프팅 (Real world prompting)](./real_world_prompting/README.md) - 복잡하고 실제적인 프롬프트에 프롬프팅 기술을 통합하는 방법을 배웁니다. [[Google Vertex 버전](https://github.com/anthropics/courses/tree/vertex/real_world_prompting)]
4. [프롬프트 평가 (Prompt evaluations)](./prompt_evaluations/README.md) - 프롬프트의 품질을 측정하기 위해 프로덕션 수준의 프롬프트 평가를 작성하는 방법을 배웁니다.
5. [도구 사용 (Tool use)](./tool_use/README.md) - Claude를 사용한 워크플로우에서 도구 사용(Tool Use)을 성공적으로 구현하는 데 필요한 모든 것을 배웁니다.

**이 과정들은 학습자의 API 비용을 낮추기 위해 가장 저렴한 모델인 Claude 3 Haiku를 주로 사용합니다. 원하신다면 다른 Claude 모델을 자유롭게 사용하셔도 좋습니다.**
